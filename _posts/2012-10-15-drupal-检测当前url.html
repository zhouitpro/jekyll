--- layout: post title: Drupal 检测当前Url, path tags: - Drupal status: publish type: post published: true meta: _edit_last: '1' --- 在Drupal可以用$_GET['q']来接收你当前的Url 但是如果你这样判断$_GET['q'] == 'my/path'; 这样是不对的.用下面这种方法 

function url_check($path) {
    return drupal_match_path($_GET['q'], $path) OR drupal_match_path(drupal_get_path_alias($_GET['q']), $path);
}
