--- layout: post title: apache .htaccess配置密码访问 .htpasswd tags: - apache status: publish type: post published: true meta: _edit_last: '1' --- 在/home/zhouitpro/www目录生成密码文件, 用户名为 zhouitpro,回车后会自动叫你输密码 
#htpasswd -c .htpasswd  zhouitpro
配置.htaccess 

AuthType Basic
AuthName "Restricted Files"
# (Following line optional)
AuthBasicProvider file
AuthUserFile /home/zhouitpro/www/.htpasswd 
Require valid-user
