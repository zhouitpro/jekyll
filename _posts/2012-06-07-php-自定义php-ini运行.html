--- layout: post title: php 自定义php.ini运行 tags: - PHP status: publish type: post published: true meta: _edit_last: '1' --- 根据自定义的php.ini来运行特定的php文件 
#php -c my_php.ini ./demo.php
使用-d来自定义php.ini中的参数,后运行php。demo.php中代码为 
#!/usr/bin/php
<?php
echo ini_get('max_file_uploads')."\n";

#php -d max_file_uploads=80  demo.php
运行结果是 一般来说在网页中是无法改变max_file_uploads这类的参数
