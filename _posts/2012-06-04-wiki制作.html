Python-sphinx 教程:

安装python-sphinx和python2-sphinx http://sphinx.pocoo.org/contents.html Arch: 
sudo pacman -S python-sphinx python2-sphinx
Ubuntu: 
sudo apt-get install python-sphinx phthon3-sphinx
1.项目创建 
$sphinx-quickstart
$cp source/* ./ -rf
2.将项目的html生成到指定的目录 
$sphinx-build -b html sourcedir builddir
3.生成html $make html 简单使用: 内嵌标记: *文字* //倾斜的字体 ``文字`` //带背景的字体 **文字** //加粗加大字符串
