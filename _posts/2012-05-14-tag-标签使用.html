--- layout: post title: Tag 标签使用 tags: - Git status: publish type: post published: true meta: _edit_last: '1' --- git的tag，跟分支一样简单。 打tag 每次上线养成打tag好习惯 创建一个tag_demo 
git tag tag_demo
覆盖已存在的tag_demo 只需要在后面加-f 
git tag -f tag_demo
显示所有的tag 
git tag
删除tag_demo tag 
git tag -d tag_demo
操作服务器上的tag 1.将创建的tag 提交到服务器上,origin是服务器的仓库,tag_demo是创建的tag 
git push origin tag_demo
2.删除服务器上的tag tag_demo是tag名,将空值提交到服务器上的tag_demo,则删除了tag_demo 
git push origin :tag_demo
3.把所有的tag都push到服务器上,此处后面--tags是参数不可改变 ,origin是仓库名 
git push origin --tags
