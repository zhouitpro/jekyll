--- layout: post title: 服务器搭建一个网站的GIT tags: - Git status: publish type: post published: true meta: _edit_last: '1' --- 在服务器上建立一个git空仓库 
# mkdir -p /var/repos/my.git     //创建git目录
# cd /var/repos/my.git
# git init --bare               //创建一个空仓库
# cd hooks
# vim post-receive              //创建钩子文件
写入以下内容 /var/www/mysite 你的网站目录 
#!/bin/sh
GIT_WORK_TREE=/var/www/mysite/ git checkout -f
还要给钩子文件加上执行权限 
# chmod +x post-receive
