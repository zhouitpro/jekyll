--- layout: post title: git log 用法 tags: - Git status: publish type: post published: true meta: _edit_last: '1' --- 查看全部提交日志 
$git log
  查看两次提交 
$git log 2
  查看提交日志，以图形方式显示,刚刚发现的超好用~~~~~~~~~ 
$git log –graph
  显示所有的提交, 只显示提交 
$git log --oneline --decorate
查看某个文件的修改历史 
$git log --pretty=oneline 文件名
显示某次提交修改的文件 
$git log --stat -2 查看近两次提交修改的文件
$git log --stat 4a12bde1403d 显示这个哈希以前的修改历史
